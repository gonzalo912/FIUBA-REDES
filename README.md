# Redes

Repositorio con los trabajos prácticos en los que participé para la materia de Redes de 4to año de la carrera de Ingeniería Informática de la Facultad de Ingeniería de la Universidad de Buenos Aires.

## TP1

La consigna fue crear un protocolo RDT que corra sobre UDP para transferir archivos binarios.

## TP2

Programar el control de un switch, respetando la jerarquía de una SDN, donde el switch obedece a las reglas instaladas por el controlador.
